# Pointer - Buitenbeentjes

This project is about a datavis made in D3.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To run in hot module reloading mode:

```sh
npm start
```


## Running

```sh
node dist/bundle.js
```
